```powershell
pip install -r requirements.txt

- GitPython
- python-dotenv
- requests
- colorama

```powershell

```

- LLM Mode: `api` (uses Ollama)
- API URL: `http://localhost:11434/v1`
- Model: `openchat:7b`

```powershell

# Stage it

# Generate commit message
python main.py --dry-run
```

**Expected output:**
